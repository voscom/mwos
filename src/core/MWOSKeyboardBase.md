# MWOSKeyboardBase

**Имя модуля:** `keyboard`

**MWOSKeyboardBase** — базовый класс для организации работы с клавиатурой в системе MWOS. Поддерживает обработку комбинаций нажатых клавиш, отслеживание времени нажатия и буферизацию событий клавиатуры.

## Описание

Класс предназначен для:
*   **Обработка клавиатуры:** Приём и обработка нажатий клавиш с поддержкой комбинаций.
*   **Буферизация событий:** Хранение списка нажатых клавиш с разделением на текущие и новые.
*   **Отслеживание времени:** Фиксация времени нажатия с момента первой кнопки комбинации.
*   **Управление событиями:** Возможность удаления нажатых клавиш из обработки другими модулями.

**Наследование:** `MWOSKeyboardBase` → `MWOSModule`

**Тип модуля:** `MODULE_KEYBOARD`

**Зависимости:**
*   Библиотека `MWTimeout` (таймауты)
*   Платформа: Любая совместимая с Arduino (ESP32, STM32, AVR и др.)

## Основные возможности

*   **Многокнопочные комбинации:** Поддержка до `MWOS_KEYBOARD_BUFFER` одновременно нажатых клавиш.
*   **Разделение событий:** Отдельное хранение текущих нажатых и новых клавиш фрейма.
*   **Таймер нажатия:** Отсчёт времени с момента нажатия первой кнопки комбинации.
*   **Удаление клавиш:** Возможность удалить клавишу из обработки, чтобы другие модули её не поймали.
*   **Получение кодов:** Методы для получения кода символа первой нажатой клавиши.
*   **Проверка состояния:** Методы проверки нажатия любой или конкретной клавиши.

## Параметры конфигурации

Класс использует параметры базовых классов и добавляет свои специфические параметры:

| ID | Название | Тип | Группа | Хранилище | Кол-во | Описание |
|:--:|:---|:---|:---|:---|:---:|:---|
| 0 | `keys` | `string` | `control` | `NO` | `MWOS_KEYBOARD_BUFFER` | **Коды нажатых клавиш.** Список кодов символов нажатых клавиш в данный момент |

**Примечание:**
*   **Кол-во:** Параметр `keys` имеет `MWOS_KEYBOARD_BUFFER` значений (по умолчанию 20).
*   **Хранилище:** Параметр не сохраняется в энергонезависимой памяти (RAM).
*   **Буфер:** Размер буфера задаётся макросом `MWOS_KEYBOARD_BUFFER` перед подключением заголовка.

*   **Настройка размера буфера:** Через макрос `#define MWOS_KEYBOARD_BUFFER 20` перед подключением.

## Основные методы

### Чтение значений

*   `virtual int64_t getValue(MWOSParam * param, int16_t arrayIndex)` *(переопределён)*
    *   Возвращает значение параметра модуля.
    *   **Возвращает:** `int64_t` — код символа клавиши по индексу.

### Инициализация и обработка

*   `virtual void onEvent(MWOSModeEvent modeEvent)` *(переопределён)*
    *   **`EVENT_UPDATE`:** Очистка списка новых клавиш `_keysNew` для следующего фрейма.

*   `virtual void onReceiveValue(MWOSNetReceiverFields * receiverDat)` *(переопределён)*
    *   Обработка входящих команд для параметра `keys`.
    *   Вызывает `pressKey()` при получении кода клавиши.

### Управление клавишами

*   `bool IsKeyPress()`
    *   Проверяет, была ли нажата любая новая клавиша в этом фрейме.
    *   **Возвращает:** `bool` — `true` если есть новые нажатия.

*   `bool IsKeyPress(char k)`
    *   Проверяет, была ли нажата конкретная клавиша в этом фрейме.
    *   **Возвращает:** `bool` — `true` если клавиша нажата.

*   `bool IsKeyPressedNow(char k)`
    *   Проверяет, нажата ли клавиша сейчас (включая предыдущие фреймы).
    *   **Возвращает:** `bool` — `true` если клавиша нажата.

*   `void RemoveKey(char k)`
    *   Удаляет код клавиши из текущего фрейма, чтобы другие модули его не обработали.
    *   **Параметры:** `k` — код клавиши для удаления.

*   `void RemoveKeys()`
    *   Удаляет все клавиши из текущего фрейма.

*   `char GetPressedKey()`
    *   Возвращает код символа первой нажатой клавиши в этом фрейме.
    *   **Возвращает:** `char` — код символа или 0 если нет нажатых.

## Пример использования

### Базовое подключение

```cpp
#include "MWOS3.h"
#include "MWOSKeyboardBase.h"

// Создаём экземпляр модуля клавиатуры
MWOSKeyboardBase keyboard;

void setup() {
    mwos.start();
}

void loop() {
    mwos.update();
    
    // Проверка нажатия клавиш
    if (keyboard.IsKeyPress()) {
        char key = keyboard.GetPressedKey();
        // Обработка нажатой клавиши
    }
}
```

### Обработка комбинаций клавиш

```cpp
void loop() {
    mwos.update();
    
    // Проверка конкретной клавиши
    if (keyboard.IsKeyPress('A')) {
        // Клавиша A нажата в этом фрейме
    }
    
    // Проверка удержания клавиши
    if (keyboard.IsKeyPressedNow('B')) {
        // Клавиша B удерживается
    }
    
    // Удаление клавиши из обработки
    keyboard.RemoveKey('C');
}
```

## Хранение данных

| Параметр | Хранилище | Комментарий |
|:---|:---|:---|
| `keys` | `NO` | Не сохраняется, буфер в RAM |

**Примечание:**
*   **Буфер:** Все данные о нажатых клавишах хранятся в оперативной памяти.
*   **Сброс:** При перезагрузке все данные о нажатиях теряются.
*   **Таймер:** `timeoutPressed` также хранится в RAM и сбрасывается при отпускании всех клавиш.

## Важные замечания

1.  **Размер буфера:** По умолчанию `MWOS_KEYBOARD_BUFFER = 20`. Можно изменить через макрос перед подключением заголовка.
2.  **Фреймовая обработка:** Новые клавиши (`_keysNew`) очищаются в начале каждого фрейма `EVENT_UPDATE`.
3.  **Текущие клавиши:** Список `_keysPressed` хранит все нажатые клавиши до их отпускания.
4.  **Коды символов:** Клавиши хранятся как коды символов (ASCII). Для матричных клавиатур требуется преобразование.
5.  **Удаление клавиш:** Метод `RemoveKey()` позволяет модулю "забрать" клавишу, чтобы другие модули её не обработали.
6.  **Таймер нажатия:** `timeoutPressed` запускается при нажатии первой клавиши комбинации и останавливается при отпускании всех.
7.  **Таблица кодов:** Параметр `_codes` может использоваться для преобразования номеров клавиш в символы (PROGMEM).
8.  **Наследование:** Класс предназначен для наследования конкретными реализациями клавиатур (матричные, USB и др.).
9.  **События:** При получении команды на параметр `keys` вызывается `pressKey()` с кодом клавиши.
10. **Проверка нажатия:** `IsKeyPress()` проверяет новые нажатия, `IsKeyPressedNow()` — текущее состояние.
11. **Получение клавиши:** `GetPressedKey()` возвращает первую новую клавишу и не удаляет её из буфера.
12. **Очистка буфера:** `RemoveKeys()` удаляет все новые клавиши из текущего фрейма.
