# MWOSTimeDS3231

**Имя модуля:** `timeDS3231`

**MWOSTimeDS3231** — модуль учёта времени для работы с RTC часами реального времени DS3231/DS1307 через интерфейс I2C. Обеспечивает энергонезависимое хранение времени и точный учёт времени при отключении питания.

## Описание

Класс предназначен для:
*   **Работа с RTC DS3231:** Подключение модуля часов реального времени DS3231 или DS1307 через I2C
*   **Энергонезависимое время:** Сохранение времени при отключении питания контроллера
*   **Синхронизация времени:** Автоматическая загрузка времени из RTC при старте и сохранение при изменении
*   **Коррекция хода:** Поддержка регистра коррекции для точной настройки хода часов

**Наследование:** `MWOSTimeDS3231` → `MWOSTime` → `MWOSModule` + `MWOSDevI2C`

**Тип модуля:** `MODULE_TIME`

**Зависимости:**
*   Библиотека `DateTime.h` (для работы с датой и временем)
*   Библиотека `MWOSDevI2C.h` (для работы с I2C устройствами)
*   Платформа с поддержкой I2C (ESP32/ESP8266/STM32/Arduino)

## Основные возможности

*   **Энергонезависимое хранение времени:** RTC сохраняет время при отключении питания контроллера
*   **Автоматическая загрузка:** Время загружается из RTC при инициализации модуля
*   **Автоматическое сохранение:** Время сохраняется в RTC при изменении через сеть или программно
*   **Регистр коррекции:** Поддержка коррекции хода часов для повышения точности
*   **Регистр управления:** Доступ к регистру управления DS3231 для настройки функций
*   **I2C интерфейс:** Подключение через стандартную шину I2C (адрес 0x68)
*   **Совместимость:** Поддержка DS3231 и DS1307 (совместимые по протоколу)

## Параметры конфигурации

Класс использует параметры базовых классов MWOSTime и добавляет свои специфические параметры:

| ID | Название | Тип | Группа | Хранилище | Кол-во | Описание |
|:--:|:---|:---|:---|:---|:---:|:---|
| 0 | `time` | uint64 | realtime+option | NO | 1 | **Время Unix.** Время в секундах от 1 января 1970 по Гринвичу |
| 1 | `uptime` | uint32 | realtime+option | NO | 1 | **Время работы.** Время в секундах от старта контроллера |
| 2 | `restarts` | uint16 | option+readonly | RTC | 1 | **Количество перезагрузок.** Счётчик перезагрузок контроллера |
| 3 | `freemem` | uint32 | option+readonly | NO | 1 | **Свободная память.** Текущий объём свободной памяти в байтах |
| 4 | `time24` | uint32 | realtime+option | NO | 1 | **Время от 2024.** Время в секундах от 1 января 2024 по Гринвичу |
| 5 | `timezone` | int8 | option | EEPROM | 1 | **Часовой пояс.** Смещение от Гринвича в четвертях часа (×15 мин) |
| 10 | `correction` | int8 | option | NO | 1 | **Регистр коррекции.** Коррекция хода RTC DS3231 |
| 11 | `control` | uint8 | option | NO | 1 | **Регистр управления.** Регистр управления DS3231 (0x0E) |

**Примечание:**
*   **Кол-во:** Все параметры имеют 1 значение (глобальные для системы).
*   **Параметр 5 (timezone):** Формат: значение × 15 минут. Например, 12 = 180 минут = +3:00 (Москва).
*   **Параметры 10-11:** Специфичные для DS3231, позволяют читать/записывать регистры RTC напрямую.

## Конфигурация модуля

```
; модуль timeDS3231 - учёт времени с RTC DS3231 через I2C
; 0 - Время Unix (секунды от 1 января 1970 по Гринвичу)
timeDS3231:time = 1704067200
; 1 - Время работы контроллера (секунды от старта)
timeDS3231:uptime = 0
; 2 - Количество перезагрузок контроллера
timeDS3231:restarts = 1
; 3 - Свободная память в байтах
timeDS3231:freemem = 150000
; 4 - Время от начала 2024 года (секунды)
timeDS3231:time24 = 0
; 5 - Часовой пояс (смещение в четвертях часа, 12 = +3:00 МСК)
timeDS3231:timezone = 12
; 10 - Регистр коррекции хода RTC DS3231
timeDS3231:correction = 0
; 11 - Регистр управления DS3231 (0x0E)
timeDS3231:control = 28
```

**Примечание:**
*   **Кол-во значений:** Все параметры имеют по 1 значению.
*   **Единицы измерения:** `time`, `uptime`, `time24` — секунды. `timezone` — четверти часа (15 минут). `freemem` — байты.
*   **Адрес I2C:** DS3231 использует адрес 0x68 на шине I2C.
*   **Только чтение:** Параметры `uptime`, `freemem` вычисляются автоматически и не сохраняются.

## Конструкторы

Класс не является шаблонным (одиночный модуль времени).

1.  **По умолчанию**
    ```cpp
    MWOSTimeDS3231(TwoWire *wireInstance = &Wire);
    ```
    Создаёт модуль времени с указанием шины I2C. По умолчанию используется Wire. Адрес DS3231 устанавливается в 0x68.

## Основные методы

### Чтение значений

*   `virtual int64_t getValue(MWOSParam * param, int16_t arrayIndex)` *(переопределён)*
    *   Возвращает значение запрошенного параметра.
    *   Для параметров 10-11 читает регистры DS3231 через I2C.
    *   **Возвращает:** `int64_t` — значение параметра.

*   `bool loadTimeFromRTC()`
    *   Загружает текущее время из RTC DS3231.
    *   Читает 7 байт из регистров времени DS3231.
    *   Преобразует BCD формат в двоичное время.
    *   **Возвращает:** `bool` — true при успешной загрузке.

*   `void saveTimeToRTC()`
    *   Сохраняет текущее время в RTC DS3231.
    *   Записывает 8 байт в регистры времени DS3231.
    *   Сбрасывает флаг OSF (Oscillator Stop Flag) в регистре статуса.

### Управление временем

*   `void setTime(uint64_t unixTime)` *(унаследован)*
    *   Установить время Unix.
    *   **Параметры:** `unixTime` — время в секундах от 1 января 1970.

*   `void setTime24(uint64_t time24)` *(унаследован)*
    *   Установить время от начала 2024 года.
    *   **Параметры:** `time24` — время в секундах от 1 января 2024.

*   `bool IsTimeSetting()` *(унаследован)*
    *   Проверить, установлено ли время.
    *   **Возвращает:** `bool` — true если время установлено.

### Инициализация

*   `virtual void onEvent(MWOSModeEvent modeEvent)` *(переопределён)*
    *   **`EVENT_INIT`:** Загрузка времени из RTC DS3231 при старте.
    *   **`EVENT_CHANGE`:** Обработка команд установки времени.
    *   **`EVENT_GET_VALUE`:** Возврат текущих значений параметров времени и регистров RTC.

*   `virtual void onReceiveValue(MWOSNetReceiverFields * receiverDat)` *(переопределён)*
    *   Обработка полученных значений от сервера.
    *   При изменении времени (параметр 0 или 4) сохраняет время в RTC.
    *   При изменении параметров 10-11 записывает значения в регистры DS3231.

## Пример использования

### Базовое подключение

```cpp
#include "core/MWOSTimeDS3231.h"

// Модуль времени с RTC DS3231 (шина Wire по умолчанию)
MWOSTimeDS3231 timeModule;

void setup() {
    mwos.start();
}

void loop() {
    mwos.update();
    
    // Время автоматически загружается из RTC при старте
    // и сохраняется при изменении через сеть
}
```

### Подключение к альтернативной шине I2C

```cpp
#include "core/MWOSTimeDS3231.h"

#ifdef ESP32
// Модуль времени с RTC DS3231 на шине Wire1
MWOSTimeDS3231 timeModule(&Wire1);

void setup() {
    // Инициализация альтернативной шины I2C
    Wire1.begin(21, 22); // SDA=21, SCL=22 для ESP32
    
    mwos.start();
}

void loop() {
    mwos.update();
}
#endif
```

### Чтение регистров DS3231

```cpp
#include "core/MWOSTimeDS3231.h"

// Модуль времени с RTC DS3231
MWOSTimeDS3231 timeModule;

void setup() {
    mwos.start();
}

void loop() {
    mwos.update();
    
    // Чтение регистра коррекции (параметр 10)
    int8_t correction = timeModule.getValueByParamId(10, 0);
    
    // Чтение регистра управления (параметр 11)
    uint8_t control = timeModule.getValueByParamId(11, 0);
}
```

### Установка времени вручную

```cpp
#include "core/MWOSTimeDS3231.h"

// Модуль времени с RTC DS3231
MWOSTimeDS3231 timeModule;

void setup() {
    mwos.start();
    
    // Установка времени Unix (1 января 2024 00:00:00)
    timeModule.setTime(1704067200);
    
    // Время автоматически сохранится в RTC DS3231
}

void loop() {
    mwos.update();
}
```

## Хранение данных

| Параметр | Хранилище | Комментарий |
|:---|:---|:---|
| `time` | NO | Вычисляемое значение, не хранится (хранится в RTC) |
| `uptime` | NO | Вычисляемое значение, не хранится |
| `restarts` | RTC | Счётчик перезагрузок, сохраняется в энергонезависимой памяти |
| `freemem` | NO | Вычисляемое значение, не хранится |
| `time24` | NO | Вычисляемое значение, не хранится (хранится в RTC) |
| `timezone` | EEPROM | Часовой пояс, сохраняется |
| `correction` | NO | Регистр DS3231, хранится в RTC чипе |
| `control` | NO | Регистр DS3231, хранится в RTC чипе |

## Важные замечания

1.  **Единственный модуль времени:** В системе должен быть только один активный модуль времени. Используйте `MWOSTime::getTimeModule()` для получения ссылки на активный модуль времени.

2.  **Адрес I2C:** DS3231 использует фиксированный адрес 0x68 на шине I2C. Убедитесь, что этот адрес не занят другими устройствами.

3.  **Подключение питания RTC:** Модуль DS3231 требует подключения батареи CR2032 для сохранения времени при отключении основного питания. Без батареи время будет сбрасываться.

4.  **Автоматическая загрузка:** При инициализации (`EVENT_INIT`) модуль автоматически загружает время из RTC DS3231. Это обеспечивает правильное время сразу после старта контроллера.

5.  **Автоматическое сохранение:** При изменении времени через сеть (параметр 0 или 4) модуль автоматически сохраняет время в RTC DS3231 через метод `saveTimeToRTC()`.

6.  **Регистр коррекции:** Параметр 10 позволяет читать/записывать регистр коррекции DS3231 (адрес 0x10). Используется для точной настройки хода часов (±2 ppm).

7.  **Регистр управления:** Параметр 11 позволяет читать/записывать регистр управления DS3231 (адрес 0x0E). Значение по умолчанию 0x1C (28) включает режим 32kHz и другие функции.

8.  **Флаг OSF:** При сохранении времени в RTC автоматически сбрасывается флаг OSF (Oscillator Stop Flag) в регистре статуса (0x0F). Это указывает на нормальную работу генератора.

9.  **Формат BCD:** DS3231 использует BCD формат для хранения времени. Модуль автоматически преобразует между BCD и двоичным форматом при чтении/записи.

10. **Совместимость DS1307:** Модуль совместим с DS1307 (аналогичный протокол), но некоторые функции DS3231 (коррекция, температура) могут быть недоступны.

11. **Проверка связи:** При ошибке загрузки времени из RTC модуль записывает сообщение в лог. Проверьте подключение I2C и наличие устройства на шине.

12. **Часовой пояс:** Параметр `timezone` задаётся в четвертях часа (15 минут). Значение 12 = 180 минут = +3:00 (Москва). Значение 0 = Гринвич (UTC+0).
