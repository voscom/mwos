# MWOSHzESP32

**Имя модуля:** `hz`

**MWOSHzESP32** — регулятор частоты и скважности на базе аппаратного таймера ESP32 (LEDC). Позволяет генерировать ШИМ-сигналы с настраиваемой частотой и разрешением.

## Описание

Класс предназначен для:
*   **Генерация ШИМ:** Создание широтно-импульсной модуляции на портах ESP32
*   **Настройка частоты:** Регулировка частоты ШИМ в широком диапазоне
*   **Настройка разрешения:** Выбор разрядности ШИМ (количество бит)
*   **Множественные каналы:** Поддержка до `keysCount` независимых ШИМ-каналов

**Наследование:** `MWOSHzESP32<keysCount>` → `MWOSKey<keysCount>` → `MWOSKeyBase<keysCount>` → `MWOSModule`

**Тип модуля:** `MODULE_KEY`

**Зависимости:**
*   Платформа ESP32 (требуется для работы библиотеки ledc)
*   Библиотека `MWOSKey.h` (базовый класс ключей)

## Основные возможности

*   **Аппаратный ШИМ ESP32:** Использование встроенных таймеров LEDC для генерации ШИМ
*   **Настраиваемая частота:** Установка частоты ШИМ для каждого канала индивидуально
*   **Настраиваемое разрешение:** Выбор разрядности ШИМ (1-16 бит) для каждого канала
*   **Независимые каналы:** До `keysCount` независимых ШИМ-генераторов в одном модуле
*   **Сохранение настроек:** Все параметры сохраняются в EEPROM и восстанавливаются после перезагрузки
*   **Управление через MWOS:** Полная интеграция с системой параметров и событий MWOS

## Параметры конфигурации

Класс использует параметры базовых классов и добавляет свои специфические параметры:

| ID | Название | Тип | Группа | Хранилище | Кол-во | Описание |
|:--:|:---|:---|:---|:---|:---:|:---|
| 0 | `turn` | bits1 | control | RTC | keysCount | **Состояния ключей.** 0=выкл, 1=вкл. Восстанавливается после перезагрузки |
| 1 | `invert` | bits1 | option | EEPROM | 1 | **Инверсия.** Инвертировать логическое состояние всех ключей |
| 2 | `alwaysoff` | bits1 | option | EEPROM | 1 | **Всегда отключено.** Принудительное отключение всех ключей |
| 3 | `checkbox` | bits1 | option | EEPROM | 1 | **Режим чекбокса.** При включении одного ключа отключать остальные |
| 4 | `startTurn` | bits2 | option | EEPROM | 1 | **Стартовое состояние.** 0=восстановить;1=не трогать;2=выключено;3=включено |
| 6 | `pull` | bits3 | option | EEPROM | 1 | **Подтяжка портов.** Режим (Выход;Открытый коллектор;ШИМ;ЦАП) |
| 7 | `pin` | int16 | pin | FOR_PIN | keysCount | **Порты ключей.** Номера портов для каждого ключа |
| 30 | `hz` | uint32 | option | EEPROM | keysCount | **Частота ШИМ.** Частота в герцах для каждого канала |
| 31 | `channel` | bits4 | option | EEPROM | keysCount | **Канал ШИМ.** Номер аппаратного таймера LEDC (0-15) |
| 32 | `resolution` | bits4 | option | EEPROM | keysCount | **Разрешение ШИМ.** Разрядность в битах (1-16) |

**Примечание:**
*   **Кол-во:** Параметры с `keysCount` имеют индивидуальное значение для каждого канала. Остальные параметры — по 1 значению.
*   **Параметр 30 (hz):** Частота ШИМ в герцах. По умолчанию 490 Гц.
*   **Параметр 31 (channel):** Номер канала LEDC (0-15 на ESP32). Каждый канал использует отдельный аппаратный таймер.
*   **Параметр 32 (resolution):** Разрешение ШИМ в битах. По умолчанию 8 бит (значения 0-255). Максимум 16 бит.

## Конфигурация модуля

```
; модуль hz - регулятор частоты и скважности на ESP32
; 0 - Состояния ключей (keysCount значений, 0=выкл, 1=вкл)
hz:turn = 0, 0, 0, 0, 0
; 1 - Инвертировать значение
hz:invert = 0
; 2 - Всегда отключено
hz:alwaysoff = 0
; 3 - Режим чекбокса (0=нет, 1=выключить другие при включении этого)
hz:checkbox = 0
; 4 - Стартовое состояние (0=восстановить;1=не трогать;2=выключено;3=включено)
hz:startTurn = 0
; 6 - Подтяжка портов (0=Выход;1=Открытый коллектор;2=ШИМ;3=ЦАП)
hz:pull = 2
; 7 - Порты ключей (keysCount значений)
hz:pin = 5, 6, 7, 8, -1
; 30 - Частота ШИМ в герцах (keysCount значений)
hz:hz = 490, 490, 490, 490, 490
; 31 - Канал ШИМ (аппаратный таймер LEDC 0-15) (keysCount значений)
hz:channel = 0, 1, 2, 3, 4
; 32 - Разрешение ШИМ в битах (1-16) (keysCount значений)
hz:resolution = 8, 8, 8, 8, 8
```

**Примечание:**
*   **Кол-во значений:** Параметры `turn`, `pin`, `hz`, `channel`, `resolution` имеют `keysCount` значений (по одному на каждый канал). Остальные параметры — по 1 значению.
*   **Единицы измерения:** `hz` — герцы. `resolution` — биты (1-16). `channel` — номер таймера LEDC (0-15).
*   **Ограничения ESP32:** На ESP32 доступно 16 каналов LEDC (0-15) и 8 таймеров. Каналы 0-7 могут использовать любые таймеры.

## Конструкторы

Класс является шаблонным: `MWOSHzESP32<keysCount>`.

1.  **По умолчанию**
    ```cpp
    MWOSHzESP32();
    ```
    Создаёт модуль с параметрами по умолчанию. Все порты инициализируются значением -1 (не подключены). Частота по умолчанию 490 Гц, разрешение 8 бит.

2.  **Из массива в PROGMEM**
    ```cpp
    MWOSHzESP32(const uint8_t * pins);
    ```
    Создаёт модуль с портами из массива в памяти PROGMEM. Позволяет задать конфигурацию портов на этапе компиляции.

3.  **Одиночный пин**
    ```cpp
    MWOSHzESP32(MWOS_PIN_INT pin);
    ```
    Создаёт модуль с одним ШИМ-каналом на указанном порту. Удобно для простых конфигураций.

## Основные методы

### Чтение значений

*   `virtual int64_t getValue(MWOSParam * param, int16_t arrayIndex)` *(переопределён)*
    *   Возвращает значение запрошенного параметра.
    *   Для параметров 30-32 возвращает частоту, канал и разрешение из локальных переменных.
    *   **Возвращает:** `int64_t` — значение параметра.

### Инициализация

*   `virtual void onEvent(MWOSModeEvent modeEvent)` *(переопределён)*
    *   **`EVENT_INIT` / `EVENT_CHANGE`:** Загрузка настроек из хранилища (частота, канал, разрешение).
    *   **`EVENT_UPDATE`:** Вызывает обработку базового класса для управления ШИМ.

*   `virtual uint8_t turn(uint8_t mode, int16_t index, bool saveToStorage)` *(переопределён)*
    *   Изменяет значение ключа и настраивает ШИМ через библиотеку ledc.
    *   При включении ключа настраивает канал LEDC с заданной частотой и разрешением.
    *   При выключении ключа устанавливает значение ШИМ в 0.
    *   **Возвращает:** `uint8_t` — фактическое состояние порта с учётом инверсии.

## Пример использования

### Базовое подключение

```cpp
#include "MWOSHzESP32.h"

#ifdef ESP32
// Создаём тип для 4 ШИМ-каналов
MWOSHzESP32<4> hzModule;

// Массив пинов в PROGMEM
const uint8_t PROGMEM pins[] = {5, 6, 7, 8};

// Экземпляр модуля
hzModule(pins);

void setup() {
    // Настройки по умолчанию (загружаются только если хранилище не инициализировано)
    // hzModule._hz[0] = 1000;  // Частота 1000 Гц для первого канала
    
    // Запуск системы - загружает настройки из хранилища
    mwos.start();
}

void loop() {
    // Все действия выполняются внутри модулей через обработчики событий
    mwos.update();
}
#endif
```

### Настройка частоты и разрешения

```cpp
#include "MWOSHzESP32.h"

#ifdef ESP32
// Создаём модуль с 2 ШИМ-каналами
MWOSHzESP32<2> hzModule;

void setup() {
    // Настройка частоты для каждого канала (загружается только если хранилище не инициализировано)
    hzModule._hz[0] = 5000;     // 5 кГц для первого канала (например, для сервопривода)
    hzModule._hz[1] = 490;      // 490 Гц для второго канала (например, для двигателя)
    
    // Настройка разрешения
    hzModule._resolution[0] = 10;  // 10 бит (0-1023) для первого канала
    hzModule._resolution[1] = 8;   // 8 бит (0-255) для второго канала
    
    // Настройка каналов LEDC
    hzModule._channel[0] = 0;  // Канал LEDC 0
    hzModule._channel[1] = 1;  // Канал LEDC 1
    
    // Запуск системы
    mwos.start();
}

void loop() {
    mwos.update();
}
#endif
```

## Хранение данных

| Параметр | Хранилище | Комментарий |
|:---|:---|:---|
| `turn` | RTC | Состояния ключей, восстанавливаются после перезагрузки |
| `pin` | FOR_PIN | EEPROM или NO в зависимости от конфигурации |
| `pull` | EEPROM | Режим подтяжки портов, сохраняется |
| `hz` | EEPROM | Частота ШИМ, сохраняется |
| `channel` | EEPROM | Канал LEDC, сохраняется |
| `resolution` | EEPROM | Разрешение ШИМ, сохраняется |
| `invert`, `alwaysoff`, `checkbox` | EEPROM | Флаги режима работы, сохраняются |
| `startTurn` | EEPROM | Поведение при старте, сохраняется |

## Важные замечания

1.  **Только ESP32:** Модуль работает только на платформе ESP32. Использует библиотеку ledc, которая недоступна на других платформах.

2.  **Ограничения каналов:** На ESP32 доступно 16 каналов LEDC (0-15) и 8 таймеров. Каждый канал может использовать любой таймер, но каналы с одинаковым таймером будут иметь одинаковую частоту и разрешение.

3.  **Зависимость частоты и разрешения:** Максимальная частота ШИМ зависит от выбранного разрешения. Чем выше разрешение, тем ниже максимальная частота. Например, при 16 битах максимальная частота около 1220 Гц.

4.  **Порядок инициализации:** Все настройки по умолчанию должны задаваться **до** вызова `mwos.start()`. После запуска системы значения загружаются из хранилища, и повторная установка значений в коде будет перезаписывать сохранённые настройки при каждой перезагрузке.

5.  **Базовый класс MWOSKey:** Модуль наследуется от MWOSKey, что означает поддержку всех функций управления ключами (инверсия, чекбокс, принудительное отключение и т.д.).

6.  **Управление скважностью:** Скважность ШИМ управляется через параметр `_periodValue` базового класса MWOSKey (значения 0-255 для 8 бит, 0-1023 для 10 бит и т.д.).

7.  **Архитектура MWOS:** Все действия выполняются внутри модулей через обработчики событий. Пользовательский код в `loop()` должен содержать только вызов `mwos.update()`.

8.  **Сохранение состояния:** Состояния ключей сохраняются в RTC (энергонезависимое ОЗУ) при каждом изменении. Это обеспечивает восстановление состояния после перезагрузки без износа EEPROM.

9.  **Режим ШИМ:** Параметр `pull` должен быть установлен в 2 (ШИМ) для корректной работы модуля.

10. **Перенастройка частоты:** Изменение частоты ШИМ требует перенастройки канала LEDC. Модуль автоматически выполняет перенастройку при изменении параметра `hz`.

## См. также

*   **MWOSKey**: Базовый модуль ключей для управления портами микроконтроллера. Предоставляет физическую коммутацию и базовые функции управления.
*   **MWOSKeyBase**: Базовый класс для всех модулей ключей. Реализует общую логику управления состояниями ключей, инверсию и режим чекбокса.
*   **MWOSKeyAnalog**: Модуль аналоговых ключей с поддержкой ШИМ и ЦАП. MWOSHzESP32 использует аналогичный механизм для управления скважностью.
*   **MWOSHzSTM32**: Аналогичный модуль для платформы STM32 с использованием аппаратных таймеров.
*   **Библиотека ledc**: Встроенная библиотека ESP32 для управления ШИМ через аппаратные таймеры LEDC.
