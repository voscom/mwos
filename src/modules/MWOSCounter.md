# MWOSCounter

**Имя модуля:** `counter`

**MWOSCounter** — базовый модуль счётчика импульсов или замера частоты. Предназначен для подсчёта внешних событий или измерения временных интервалов.

## Описание

Класс предназначен для:
*   **Подсчёт импульсов:** Счёт внешних событий от энкодеров, датчиков оборотов, импульсных датчиков
*   **Измерение времени:** Работа в режиме таймера для измерения временных интервалов (текущая частота)
*   **Контроль положения:** Отслеживание позиции с учётом шага и направления
*   **Аппаратная поддержка:** Использование аппаратных счётчиков ESP32 (PCNT) или программный подсчёт

**Наследование:** `MWOSCounter` → `MWOSModule`

**Тип модуля:** `MODULE_ENCODER`

**Зависимости:**
*   Библиотека `MWOSModule.h` (базовый класс)
*   Платформа ESP32 (опционально, для аппаратного режима PCNT)

## Основные возможности

*   **Множественные режимы работы:** Программный подсчёт, на прерываниях, аппаратный счётчик, альтернативный режим (PCNT), режим таймера
*   **Учёт шага:** Преобразование сырых значений счёта в физические единицы (мм, градусы, обороты)
*   **Контроль диапазона:** Ограничение минимального и максимального значения счётчика
*   **Направление движения:** Определение направления счёта (вперёд/назад/остановлен)
*   **Целевые позиции:** Установка вызова функции по достижении заданной позиции
*   **Пауза/продолжение:** Временная остановка счёта без потери текущего значения

## Параметры конфигурации

Класс использует параметры базовых классов и добавляет свои специфические параметры:

| ID | Название | Тип | Группа | Хранилище | Кол-во | Описание |
|:--:|:---|:---|:---|:---|:---:|:---|
| 0 | `counter` | int32 | control | RTC | 1 | **Текущая позиция.** Счёт без учёта шага. Восстанавливается после перезагрузки |
| 1 | `position` | double64 | control | NO | 1 | **Позиция с шагом.** Счёт с учётом коэффициента шага |
| 2 | `status` | int8 | control | NO | 1 | **Статус счётчика.** 0=назад;1=вперёд;2=остановлен;3=на паузе |
| 3 | `step` | float32 | option | EEPROM | 1 | **Шаг энкодера.** Коэффициент преобразования счёта в физические единицы |
| 4 | `minimum` | int32 | option | EEPROM | 1 | **Минимум счётчика.** Нижняя граница диапазона |
| 5 | `maximum` | int32 | option | EEPROM | 1 | **Максимум счётчика.** Верхняя граница диапазона |
| 6 | `sendTimeDSec` | uint16 | option | EEPROM | 1 | **Таймаут отправки.** Период отправки значений на сервер [сек/10] |
| 7 | `type` | bits3 | option | EEPROM | 1 | **Тип счётчика.** 0=Software;1=Interrupt;2=Hardware;3=HardwareAlter;4=Timeout |

**Примечание:**
*   **Кол-во:** Все параметры имеют 1 значение (одиночный счётчик).
*   **Параметр 7 (type):** Формат значений: "Software;Interrupt;Hardware;HardwareAlter;Timeout"
*   **Параметр 2 (status):** Формат значений: "Назад;Вперёд;Остановлен;На паузе"

## Конфигурация модуля

```
; модуль counter - счётчик импульсов или времени
; 0 - Текущая позиция счёта (без учёта шага)
counter:counter = 0
; 1 - Позиция с учётом шага (физические единицы)
counter:position = 0.0
; 2 - Статус счётчика (0=назад;1=вперёд;2=остановлен;3=на паузе)
counter:status = 2
; 3 - Шаг энкодера (коэффициент преобразования)
counter:step = 1.0
; 4 - Минимум счётчика (нижняя граница)
counter:minimum = -2147483648
; 5 - Максимум счётчика (верхняя граница)
counter:maximum = 2147483648
; 6 - Таймаут отправки значений на сервер [сек/10]
counter:sendTimeDSec = 10
; 7 - Тип счётчика (0=Software;1=Interrupt;2=Hardware;3=HardwareAlter;4=Timeout)
counter:type = 0
```

**Примечание:**
*   **Кол-во значений:** Все параметры имеют по 1 значению.
*   **Единицы измерения:** `sendTimeDSec` — десятые доли секунды. `step` — физические единицы на 1 импульс.
*   **Диапазон счётчика:** `minimum` и `maximum` задают границы допустимых значений счётчика.

## Конструкторы

Класс не является шаблонным (одиночный счётчик).

1.  **По умолчанию**
    ```cpp
    MWOSCounter();
    ```
    Создаёт модуль с параметрами по умолчанию. Тип счётчика устанавливается в программный режим.

2.  **С указанием типа**
    ```cpp
    MWOSCounter(MWOSCounterType type);
    ```
    Создаёт модуль с указанным типом счётчика (Software, Interrupt, Hardware, HardwareAlter, Timeout).

## Основные методы

### Чтение значений

*   `int32_t getCounter()`
    *   Возвращает текущее значение счётчика без учёта шага.
    *   **Возвращает:** `int32_t` — сырое значение счёта.

*   `double getPosition()`
    *   Возвращает позицию с учётом шага.
    *   **Возвращает:** `double` — позиция в физических единицах.

*   `int8_t getStatus()`
    *   Возвращает текущий статус счётчика.
    *   **Возвращает:** `int8_t` — статус (0=назад;1=вперёд;2=остановлен;3=на паузе).

### Управление счётом

*   `void StartCount(int32_t position)`
    *   Сбросить позицию и начать отсчёт.
    *   **Параметры:** `position` — начальная позиция.

*   `void StartPos(double newPosition)`
    *   Установить позицию с учётом шага.
    *   **Параметры:** `newPosition` — новая позиция в физических единицах.

*   `void Pause()`
    *   Приостановить отсчёт. Статус устанавливается в "на паузе".

*   `void Resume()`
    *   Снять с паузы и восстановить нормальный режим счёта.

*   `void setPosEvent(int32_t pos, void (*callback)())`
    *   Установить вызов функции по достижении позиции.
    *   **Параметры:** `pos` — целевая позиция, `callback` — функция обратного вызова.

### Инициализация

*   `virtual void onEvent(MWOSModeEvent modeEvent)` *(переопределён)*
    *   **`EVENT_INIT` / `EVENT_CHANGE`:** Загрузка настроек из хранилища, инициализация счётчика.
    *   **`EVENT_UPDATE`:** Вызов обработки счёта (чтение импульсов, обновление статуса).

## Пример использования

### Базовое подключение

```cpp
#include "modules/encoder/MWOSCounter.h"

// Создаём счётчик
MWOSCounter counter;

void setup() {
    // Настройки по умолчанию (загружаются только если хранилище не инициализировано)
    // counter._step = 0.1;  // Шаг 0.1 мм на импульс
    
    // Запуск системы - загружает настройки из хранилища
    mwos.start();
}

void loop() {
    // Все действия выполняются внутри модулей через обработчики событий
    mwos.update();
}
```

### Подключение с аппаратным счётчиком (ESP32)

```cpp
#include "modules/encoder/MWOSCounter.h"

#ifdef ESP32
// Создаём счётчик с аппаратной поддержкой PCNT
MWOSCounter counter(CT_HARDWARE_ALTER);

void setup() {
    // Настройка шага энкодера
    counter._step = 0.05;  // 0.05 мм на импульс
    
    // Настройка диапазона
    counter._minimum = 0;
    counter._maximum = 10000;
    
    // Запуск системы
    mwos.start();
}

void loop() {
    mwos.update();
    
    // Чтение позиции
    double pos = counter.getPosition();
}
#endif
```

### Установка целевой позиции

```cpp
#include "modules/encoder/MWOSCounter.h"

// Создаём счётчик
MWOSCounter counter;

// Функция обратного вызова
void onTargetReached() {
    // Достигнута целевая позиция
}

void setup() {
    // Установка вызова по достижении позиции 1000
    counter.setPosEvent(1000, onTargetReached);
    
    // Запуск системы
    mwos.start();
}

void loop() {
    mwos.update();
}
```

### Управление паузой

```cpp
#include "modules/encoder/MWOSCounter.h"

// Создаём счётчик
MWOSCounter counter;

void setup() {
    // Запуск системы
    mwos.start();
}

void loop() {
    mwos.update();
    
    // Проверка состояния
    if (counter.getStatus() == 3) {
        // Счётчик на паузе
    }
    
    // Постановка на паузу
    counter.Pause();
    
    // Снятие с паузы
    counter.Resume();
}
```

## Хранение данных

| Параметр | Хранилище | Комментарий |
|:---|:---|:---|
| `counter` | RTC | Текущее значение счёта, восстанавливается после перезагрузки |
| `step` | EEPROM | Шаг энкодера, сохраняется |
| `minimum`, `maximum` | EEPROM | Границы диапазона, сохраняются |
| `sendTimeDSec` | EEPROM | Таймаут отправки, сохраняется |
| `type` | EEPROM | Тип счётчика, сохраняется |
| `position` | NO | Вычисляемое значение, не хранится |
| `status` | NO | Текущий статус, не хранится |

## Важные замечания

1.  **Типы счётчиков:** Доступны 5 режимов работы:
    *   0 = Software (программный подсчёт)
    *   1 = Interrupt (на прерываниях)
    *   2 = Hardware (аппаратный счётчик)
    *   3 = HardwareAlter (альтернативный режим, PCNT на ESP32)
    *   4 = Timeout (режим таймера для измерения времени)

2.  **Порядок инициализации:** Все настройки по умолчанию должны задаваться **до** вызова `mwos.start()`. После запуска системы значения загружаются из хранилища, и повторная установка значений в коде будет перезаписывать сохранённые настройки при каждой перезагрузке.

3.  **Шаг энкодера:** Параметр `step` задаёт коэффициент преобразования сырого счёта в физические единицы. Например, 0.1 = 0.1 мм на импульс.

4.  **Контроль диапазона:** Параметры `minimum` и `maximum` ограничивают диапазон значений счётчика. При достижении границ счётчик останавливается.

5.  **Режим паузы:** При установке режима паузы (`Pause()`) счётчик останавливается, но текущее значение сохраняется. Для возобновления счёта используйте `Resume()`.

6.  **Архитектура MWOS:** Все действия выполняются внутри модулей через обработчики событий. Пользовательский код в `loop()` должен содержать только вызов `mwos.update()`.

7.  **Сохранение состояния:** Значение счётчика сохраняется в RTC (энергонезависимое ОЗУ) при каждом изменении. Это обеспечивает восстановление состояния после перезагрузки без износа EEPROM.

8.  **Аппаратная поддержка:** На ESP32 доступен режим HardwareAlter (PCNT) для высокоточного подсчёта импульсов без нагрузки на процессор.

9.  **Режим таймера:** При type=4 счётчик работает как таймер для измерения временных интервалов вместо подсчёта импульсов.

10. **Целевые позиции:** Метод `setPosEvent()` позволяет установить функцию обратного вызова, которая выполнится при достижении заданной позиции.

## См. также

*   **MWOSModule**: Базовый класс для всех модулей контроллера. Реализует общую инфраструктуру модулей, параметры и события.
*   **MWOSEncoder**: Расширение MWOSCounter для работы с энкодерами. Добавляет поддержку направления, частоты вращения и управления портами.
*   **MWOSKey**: Модуль ключей для управления портами микроконтроллера. Может использоваться вместе со счётчиком для управления исполнительными механизмами.
*   **MWOSConsts**: Константы и перечисления системы MWOS, включая типы модулей и событий.
