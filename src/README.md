## Сетевые модули
| Класс, документация                               | Описание                                                                        |
|:--------------------------------------------------|:--------------------------------------------------------------------------------|

| [MWOSNetSerial](MWOSNetSerial.md)                 | Модуль сетевого подключения через последовательный порт (любой микроконтроллер) |

## Датчики (выключатели, переключатели и т.д.)

| Класс, документация                                     | Описание                                                                                                                     |
|:--------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------|
| [MWOSSensorBase](modules/sensor/core/MWOSSensorBase.md) | Базовый класс для всех датчиков (логика состояний)                                                                           |
| [MWOSSensorFilter](modules/sensor/core/MWOSSensorFilter.md)            | Расширения аналоговых датчиков с поддержкой фильтрации, калибровки и прогнозирования значений                                |
| [MWOSSensorBin](modules/sensor/MWOSSensorBin.md)                       | Бинарные датчики: Кнопки, герконы, концевые выключатели, PIR-сенсоры.                                                        |
| [MWOSSensorADC](modules/sensor/MWOSSensorADC.md)                       | **Аналоговые датчики:** Датчики температуры, освещённости, напряжения, тока и др                                             |
| [MWOSSensorSerial](modules/sensor/MWOSSensorSerial.md)                 | **Потоковые датчики:** Работа с датчиками, передающими данные в текстовом формате (LuminOx, CO2-датчики, метеостанции и др.) |
| [MWOSSensorADS1115](modules/sensor/MWOSSensorADS1115.md)               | **Высокоточные АЦП:** 16-битный АЦП ADS1115 обеспечивает высокую точность.                                                   |
| [MWOSSensorDS](modules/sensor/MWOSSensorDS.md)                         | **Температурные датчики** семейства DS18x20.                                                                                 |
| [MWOSSensorDHT](modules/sensor/MWOSSensorDHT.md)                       | **Температура и влажность:** DHT11, DHT22, DHT33, DHT44, AM2301, AM2302, AM2303.                                                                                                |
